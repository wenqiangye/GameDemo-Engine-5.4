# GameDemo-Engine-5.4

# 一、基于UE5（5.4）的一个demo

- 角色的基本运动（动画状态机）
- 角色健康组件、武器组件、移动组件
- 健康组件：脱离战斗自动回血，角色伤害计算
- 武器组件：目前实现了步枪武器，后续设计sword
- 移动组件：用于动画状态机中方向和速度的判断
- UI：实现了设计准星和血条
- AI：待做
- VFX：待做
